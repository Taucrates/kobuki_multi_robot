- Instalar ROS y paquetes en la NUC del kobuki ya montado
	+	Instalar melodic desktop
	+	Set up entorno catkin http://wiki.ros.org/ROS/Tutorials/InstallingandConfiguringROSEnvironment
	+	Instalar paquetes kobuki (ver mail Alberto)
	+	Activar conexión ssh https://linuxize.com/post/how-to-enable-ssh-on-ubuntu-18-04/
	+	sudo apt-get install ros-melodic-teb-local-planner (finalmente se ha optado por DWA y no hace falta instalar este package)
	+	sudo apt-get install ros-melodic-amcl
	+	sudo apt-get install ros-melodic-yocs-velocity-smoother

- Instalar htop

- Adicionalmente se proveen de librerias de real-sense
	- https://github.com/IntelRealSense/realsense-ros
	- Copiar .rules en /etc/udev/rules.d/ si ROS no detecta la camara por USB
		+ https://github.com/IntelRealSense/librealsense/blob/master/config/99-realsense-libusb.rules

- Gmapping >     sudo apt-get install ros-melodic-gmapping
- Navagation stack >	    sudo apt-get install ros-melodic-navigation
- Depth-to-scan >	sudo apt-get install ros-melodic-depthimage-to-laserscan
	+ rosrun depthimage_to_laserscan depthimage_to_laserscan image:=/camera/depth/image_rect_raw camera_info:/camera/depth/camera_info

- Para el hokuyo 04LX
	+ sudo adduser <YOUR_USERNAME> dialout
	+ od /dev/ttyACM0
	+ sudo apt install ros-melodic-urg-node
	+ rosrun urg_node urg_node

- Para el hokuyo 20LX
	+ La dirección por defecto del hokuyo es 192.168.0.10
	+ Configurar puerto ethernet como 192.168.0.20/25
	+ rosrun urg_node urg_node _ip_address:=192.168.0.10

- Para el RPLIDAR
	+ Descargar repositorio https//github.com/Slamtec/rplidar_ros
	+ sudo adduser <YOUR_USERNAME> dialout
	+ sudo chmod 666 /dev/ttyUSB0
	+ comprobar ls -al /dev/ttyUSB0
	+ launch rplidar_s1.launch

- En BIOS
	+ Opcion Advanced - Boot - Boot configuration - Suppress Alert Messages At Boot >> Desactivada

- Despues de configurar IP estatica ver
	ssh-keyscan -H 192.168.1.162 >> ~/.ssh/known_hosts
